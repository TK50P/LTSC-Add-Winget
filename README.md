# Add Winget for LTSC

## Usage
Run `Add-Winget.cmd` as **Administrator**.

## Supported OS
- Windows 10 Enterprise LTSC 2019 / 2021
- Windows 11 Enterprise LTSC 2024
- Windows 10 / 11 IoT Enterprise LTSC

## Minimum Requirements
- Windows 10 Version 1709 (Build 16299) or later

## Notes
- Microsoft Store **will not needed/installed**
- This script installs **App Installer (winget)** and required dependencies only
- All `.Appx` / `.Msix` files must be in the same directory as `Add-Winget.cmd`
- Reboot may be required after installation
